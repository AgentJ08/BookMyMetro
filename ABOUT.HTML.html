ABOUT US

The current system of booking metro is tedious and non practical as it wastes a lot of time and energy. A system is required which can change the orientation of this scenario while also enabling user with other essential features of metro.The objective of this project is to develop a system which enables users to avoid long queues and book metro tokens online with just a single click. A system which is currently non-existent .This will save a lot of time and energy. A user will have complete access to the following features:-

• Book metro token
• Get shortest metro route and fare information
• Easy and secure digital payment
• Latest metro updates
• Complaint and feedback portal
• Access to complete metro network
